# Day 06 – Object Oriented Programming (OOP)

## Topics Covered
- Classes and Objects
- Constructors (`__init__`)
- Inheritance
- Polymorphism
- Encapsulation
- Real-world OOP examples

## Files
- `oop_basics.py` – Class & object basics

- `polymorphism.py` – Method overriding & flexibility
- `encapsulation.py` – Data hiding and access control

## Goal
Understand OOP concepts used heavily in backend frameworks
like Django, FastAPI, and real-world Python applications.
