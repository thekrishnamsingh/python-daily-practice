# Day 03 – Functions (Advanced) & Modules

This day focuses on writing reusable functions and
organizing code using modules.

## Topics Covered
- Default arguments
- Return vs print
- Multiple return values
- *args and **kwargs
- Creating and importing custom modules

## Files
- `functions_advanced.py` – Advanced function examples
- `math_utils.py` – Custom module
- `main.py` – Importing and using modules

## Goal
Write clean, reusable Python code and understand
how real projects structure logic using modules.
